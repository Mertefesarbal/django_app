12""" 

13Django settings that affect django-autoslug: 

14 

15`AUTOSLUG_SLUGIFY_FUNCTION` 

16 Allows to define a custom slugifying function. 

17 

18 The function can be repsesented as string or callable, e.g.:: 

19 

20 # custom function, path as string: 

21 AUTOSLUG_SLUGIFY_FUNCTION = 'some_app.slugify_func' 

22 

23 # custom function, callable: 

24 AUTOSLUG_SLUGIFY_FUNCTION = some_app.slugify_func 

25 

26 # custom function, defined inline: 

27 AUTOSLUG_SLUGIFY_FUNCTION = lambda slug: 'can i haz %s?' % slug 

28 

29 If no value is given, default value is used. 

30 

31 Default value is one of these depending on availability in given order: 

32 

33 * `unidecode.unidecode()` if Unidecode_ is available; 

34 * `pytils.translit.slugify()` if pytils_ is available; 

35 * `django.template.defaultfilters.slugify()` bundled with Django. 

36 

37 django-autoslug also ships a couple of slugify functions that use 

38 the translitcodec_ Python library, e.g.:: 

39 

40 # using as many characters as needed to make a natural replacement 

41 AUTOSLUG_SLUGIFY_FUNCTION = 'autoslug.utils.translit_long' 

42 

43 # using the minimum number of characters to make a replacement 

44 AUTOSLUG_SLUGIFY_FUNCTION = 'autoslug.utils.translit_short' 

45 

46 # only performing single character replacements 

47 AUTOSLUG_SLUGIFY_FUNCTION = 'autoslug.utils.translit_one' 

48 

49.. _Unidecode: http://pypi.python.org/pypi/Unidecode 

50.. _pytils: http://pypi.python.org/pypi/pytils 

51.. _translitcodec: http://pypi.python.org/pypi/translitcodec 

52 

53`AUTOSLUG_MODELTRANSLATION_ENABLE` 

54 Django-autoslug support of modeltranslation_ is still experimental. 

55 If you wish to enable it, please set this option to `True` in your project 

56 settings. Default is `False`. 

57 

58.. _modeltranslation: http://django-modeltranslation.readthedocs.org 

59 

60"""
